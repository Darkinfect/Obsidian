   Разработать многомодульное приложение для решения следующей задачи:
 Ввести целое число. Ввод числа организовать из головного модуля на С++. Определить, является ли модуль этого числа степенью 2. Если да, то вывести эту степень (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести целое число. Ввод числа организовать из головного модуля на С++. Вычислить сумму цифр модуля этого числа (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести целое число. Ввод числа организовать из головного модуля на С++. 10.	Из записи числа выбросить все четные цифры, оставив прежним порядок остальных цифр (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести целое число. Ввод числа организовать из головного модуля на С++. Определить количество четных цифр числа (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести целое число. Ввод числа организовать из головного модуля на С++. Определить целую часть среднего арифметического цифр в записи числа (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести два целых числа. Ввод числа организовать из головного модуля на С++. Определить, является ли модуль одного из них степенью модуля второго. Если да, то вывести эту степень (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести массив целых чисел. Ввод организовать из головного модуля на С++. Найти сумму положительных чисел массива (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести массив целых чисел и число Х. Ввод организовать из головного модуля на С++. Определить, есть ли число Х в массиве (Ассемблер). Вывести результат (С++).[[Решения задач]]
 Ввести массив целых чисел. Ввод организовать из головного модуля на С++. Отсортировать массив по возрастанию (Ассемблер). Вывести результат (С++).[[Решения задач]]

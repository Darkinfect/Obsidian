## Составляющие компоненты программы
Программа в ассемблере делиться на 3 сегмента:
1. Сегмент кода
2. Сегмент данных
3. Сегмент Стека
Сегменты располагаются в разных частях памяти.
Все адреса сегментов распределяется системой и компоновщиком
Максимальным размером каждого сегмента будет являтся 64КБ
### Пример реализации нескольких сегментов
```
.model flat
.stack 100h
.data
    message db "hello", '$'
.code
  mov eax, offset db ; гружу адресс в eax
end
```

Существует два вида сдвигов
* Линейный
* Циклический
## Линейный
1. Очередной «выдвигаемый» бит устанавливает флаг

CF (сдвигается во флаг CF).

2. Бит, появляющийся с другого конца операнда, имеет

значение 0.

3. При сдвиге очередного бита он переходит во флаг

CF, при этом значение предыдущего сдвинутого бита

теряется.

### Команды
Грубо говоря Логический/Арифметический влево(равны)
Команда 
```
	Shl оп1, кол-во итераций
	(сам бит мы смещаем влево то-есть самый левый уйдёт во флаг CF)
	(а в право мы вставляем 0)
```
Логический вправо
```
Shr оп1, кол-во итераций
(тоже самое что и влево только в другую сторону)
```
Арифметический вправо
```
sar оп1, число бит
(тут мы бит левый бит перезаписываем, а правый во флаг CF)
```

#Определение Стек - это структура данных. В ассемблере стек широко используется для управления данными и внутренним состоянием программы. Работает по принципу LIFO

В основном стек используется для сохранения всех регистров и состояния процессора

>[!info] Команды для работы со стеком
>PUSH регистр - засылает значение регистра в стек(для сохранения)
>PUSH константа - засылает константу в стек
>POP регистр - Значение извлекается из стека и заноситься регистр
>POP  регистр - извлекается из стека без сохранения

Операторы преобразования длины позволяют менять длину выражений.
Для того чтобы преобразовать длину нужно воспользоваться следующими командами:
1. BYTE PTR выражение
2. WORD PTR выражение
3. DWORD PTR выражение
4. QWORD PTR выражение
(не применяется при регистровой адресации)
### Пример
```
mov eax, dword ptr ebx
; в данном примере я привёл ebx к 32 битной длине операнда
```

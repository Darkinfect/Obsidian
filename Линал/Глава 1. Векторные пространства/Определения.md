#Определение Векторное пространство - это множество элементов (называемых векторами) в котором определены операции сложения векторов и умножения вектора на скаляр из поля P и при этом выполняются все аксиомы:
>[!info] Аксиомы векторного пространства
I: a + b = b + a;
II: (a + b) + c = a + (b + c);
III: Существует вектор 0 принадлежащий V (называемый нулевым вектором), что a + 0 = a;
IV: Для каждого вектора существует противоположный вектор -а такой что а +(-а) = 0
V: (α + β)a = αa + βa; 
VI: (αβ)a = α(βa);
VII: α(a + b) = αa + αb; 
VIII: 1a = a.

#Определение Скажем, что система $\mathcal{E} = \{e_1, \ldots, e_n\}$ векторов из пространства $V$ является **линейно независимой**, если линейная комбинация $\alpha_1 e_1 + \ldots + \alpha_n e_n$ векторов из $\mathcal{E}$ может равняться нуль-вектору только в тривиальном случае, т.е. когда все $\alpha_i = 0$.
#Определение Скажем, что система  $\mathcal{E} = \{e_1, \ldots, e_n\}$ является полной, если любой вектор $\alpha_{1} \in V$ раскладывается по системе $\mathcal{E}$, т.е. $\alpha = \sum\alpha_{i}e_{i}$ 
#Определение Упорядоченная система векторов $\mathcal{E} = \langle e_1, \ldots, e_n\rangle$ называется *базисом* $V$, если $\mathcal{E}$ одновременно:
1. Полна,
2. Линейно независима.

#Определение Подсистема $B$ системы $A = \{a_1, \ldots, a_s\}$ называется *базисом* $A$, если:
1. $B$ линейно независима
2. Система $A$ линейно выражается через $B$
#Определение(Размерность пространства)  
Количество векторов в базисе пространства $V$ называется его *размерностью* и обозначается $\dim(V) < \infty$. Для $V = \{0\}$ полагаем $\dim(V) = 0$.
